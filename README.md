# Panorama Jum — Pilot Project

This pilot copy has three operational areas:

1. Main dashboard
2. Parking Automation
3. Pass & Ride

All other modules and sections are intentionally unavailable in this pilot.

## One-click start

Double-click `run-project.bat`.

The launcher starts the local API and web app, installs missing dependencies on the first run, and opens the system in a browser.

## Requirements

The client computer needs Python 3.11+ and Node.js/npm available in PATH. Internet is needed only on the first run if dependencies are not already installed.

The pilot starts with a fresh local database and creates demo data automatically.
